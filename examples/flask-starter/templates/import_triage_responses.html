What happens on import:

  • Yes responses — the (connector → target) path gets marked as intro requested in the same flow as the Mark-as-requested button on connector cards.
  • No responses — the path stays in the database (we still know the connector knows them) but its score is forced to 0 wherever it renders. Effectively de-prioritizes the path without losing data.
  • Re-importing the same file is safe. Both writes are idempotent.
