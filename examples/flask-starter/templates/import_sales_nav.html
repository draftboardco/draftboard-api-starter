    How this works

    • We never store the Scrupp output server-side — fetched leads live only in this tab until you click Import.
    • Imported leads land in Draftboard via the same /targets/import endpoint as the manual paste-URLs flow. Path data shows up after Draftboard finishes resolving each LinkedIn URL (usually a minute).
    • Re-running the same search re-imports the same URLs. Draftboard's import is idempotent — duplicate URLs no-op.
    • Free Scrupp tier: 500 leads/month, 100 max per search. Heavier usage needs a paid plan.
