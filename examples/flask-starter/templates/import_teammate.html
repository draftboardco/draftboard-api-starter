How does my teammate get the scanner?

You build the distributable once and DM it to them — Slack attachment, email, private gist. They run it on their laptop, no setup beyond Python.

  1. cd examples/flask-starter
  2. python3 scanner/build_dist.py (reads your OAuth client from ~/.draftboard-secrets/google.env)
  3. DM them scanner/dist/supporter_scan.py
  4. They run python3 supporter_scan.py, OAuth, get an HTML review page alongside the script
  5. They open the HTML in their browser, untick anyone they don't want shared, click Save Filtered JSON — a JSON file downloads
  6. They send THAT filtered JSON back to you (not the HTML)

Full instructions for them: scanner/README.md — copy the "What teammates do" section into your DM.

Important: their email needs to be on your project's test-users allowlist in Google Cloud (you have a 100-email cap), or they'll see "Access blocked" on the Google sign-in page. Add them at console.cloud.google.com → OAuth consent screen → Test users.
