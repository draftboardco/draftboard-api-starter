{# ---- Show me how to find my API key (the click-by-click) ---- #}
Show me how to find my Draftboard API key →
  1. Open intros.draftboard.com and sign in to your workspace.
  2. Click your avatar (top-right) → Settings.
  3. In the left sidebar, click API Keys.
  4. Click Generate API key. Give it a label (e.g. local-starter-kit) so you remember which install uses it.
  5. Copy the key — it starts with db-api_ and looks like a UUID. It's only shown once. If you lose it, generate a new one (and revoke the old).
  6. Paste it into the field above and click Validate & save.

The key is stored locally in ~/.draftboard-secrets/draftboard-api-starter with file permissions 0600 (owner-only). It never leaves your machine.
