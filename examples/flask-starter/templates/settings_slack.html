Want to set up Slack notifications?

Setup takes about 5 minutes:

  1. Pick a Slack channel where alerts should land
  2. Create a Slack App and add an Incoming Webhook for that channel
  3. Paste the webhook URL here
  4. Map each teammate's Slack user ID (on the Team Settings page)
  5. Send a test message to confirm

You can stop after step 1 or 2 — the channel name + webhook are saved as you go, so you can resume any time.

{# ---------- Step 2: Pick a channel ---------- #} {% elif current_step == 2 %}

Pick a Slack channel

Which channel should warm-intro alerts land in? We default to #warm-intros, but pick whatever fits your team — many customers use an existing #sales channel, a dedicated team channel, or a private channel. The name you save here is just a label that shows up in this app's UI; it just has to match the channel where you'll install the webhook in the next step.

The # is added automatically if you skip it.

Don't have a channel for this yet? In Slack: click the + next to Channels → Create channel → name it whatever you like (most teams pick something descriptive like #warm-intros or #sales-intros) → set it private if only the sales team should see the alerts → Create. Then come back here and type the same name above.

Already have a #warm-intros channel that does something else? No problem — just type whatever name you want above. The app uses this label only as a UI hint; what actually routes messages to Slack is the webhook URL in the next step.

Install a Slack Incoming Webhook

A webhook is a one-way URL that Slack gives you so this app can post messages into the channel — no bot, no OAuth, no extra permissions. Takes ~2 minutes.

  1. Open api.slack.com/apps and sign in to your workspace.
  2. Click Create New App → From scratch.
  3. Name it Draftboard Alerts (or anything you like) and pick your workspace → Create App.
  4. In the left sidebar, click Incoming Webhooks.
  5. Toggle Activate Incoming Webhooks to On.
  6. Scroll down, click Add New Webhook to Workspace.
  7. Pick the {{ channel_name or '#warm-intros' }} channel from the dropdown → Allow.
  8. You'll land back on the Incoming Webhooks page. Copy the new webhook URL. It looks like https://hooks.slack.com/services/T0…/B0…/….

Slack restricts who can install apps. If you're not the workspace owner / admin, you may need to ask one to approve "Draftboard Alerts" before step 6 above will work for you.
