CSV format we accept

  • First row must be a header. We try header names first ("LinkedIn URL", "profile", "li_url", etc.) then sniff cell content for linkedin.com/in/.
  • Each row needs at least a LinkedIn URL. Bare slugs like zacharyroseman work; we'll canonicalize to https://www.linkedin.com/in/zacharyroseman.
  • Other columns are ignored except for a "Name" column (used as display name on supporter rows).
  • Excel-exported CSVs with a BOM marker work. UTF-8 is preferred but most encodings will read.
  • Duplicates are deduped before import.
