How the engine works

  1. gpt-4o-mini reads your ICP description + customer examples → returns a structured Apollo search (keywords, employee bands, locations) + a scoring rubric.
  2. Apollo's mixed_companies/search returns ~25 candidate companies.
  3. gpt-4o-mini scores each candidate 1-10 against the rubric and writes a one-sentence rationale.
  4. You see the scored list. Open any LinkedIn link to verify, then run Sales Nav import on the company to add people from it as Draftboard targets.

v1 doesn't auto-import accounts as Draftboard targets — Draftboard targets are people, not companies. Use the candidate list to pick promising accounts, then run the people-prospecting flow on each.
